快乐猴 RDC 候选点生成全流程（北京 / 杭州落地版）
一、先明确：社区超市 RDC 的核心选址要求（和电商 / 工业 RDC 完全不同）
社区超市 RDC 是城市末端配送中心，核心服务半径 30-50 公里，高频小批量配送（每周 2 次），优先满足生鲜 2 小时达要求，因此优先级排序：
城配路网效率 > 仓库租金成本 > 干线运输便利性 > 面积大小
✅ 核心硬指标（一票否决）：
必须在城市货车通行蓝区内，24 小时允许蓝牌货车通行，黄牌货车夜间可通行
临近城市快速路 / 绕城高速出入口，到城市核心区平均车程≤30 分钟
仓库层高≥6 米，有装卸货平台，可改造为常温 + 冷藏 + 冷冻三温仓
消防等级丙二类以上，符合食品仓储安全标准
周边无居民区、学校、医院，避免扰民投诉
❌ 绝对排除：
货车限行区、核心城区内的仓库
老旧厂房、无正规产权的仓库
周边路网拥堵、进出困难的仓库
无法改造为三温仓的仓库
二、RDC 候选点生成三步法（可直接落地执行）
第一步：锁定城市核心物流节点（大区域筛选）
先从城市整体物流规划入手，找到政府划定的重点物流园区 / 物流带，这些区域是唯一合法合规、长期稳定的仓储用地，也是未来城市物流的核心节点。
1. 权威数据源（免费公开）
城市物流空间布局规划（必看！）：
北京：《北京市物流空间布局规划（2021-2035 年）》
杭州：《杭州市现代物流业发展 “十四五” 规划》
政府官网：市发改委、市交通运输局、市商务局
地图软件：高德 / 百度地图搜索 “物流园区”“仓储中心”

第二步：筛选符合要求的物流园区（中观筛选）
从第一步锁定的物流带中，筛选出满足以下条件的园区：
园区内有5000-20000 平米的可租赁仓库（社区超市 RDC 标准面积：1 万平米可服务 50-80 家门店）
园区有正规的物流运营资质，消防、环保手续齐全
园区出入口直接连接城市快速路，无拥堵点
园区内有足够的货车停车位和装卸货场地
快速筛选方法：
打电话给园区招商部，直接问：“有没有 1 万平米左右的丙二类仓库可租？能不能改三温仓？货车通行政策怎么样？”
排除租金过高（北京＞1.8 元 /㎡/ 天，杭州＞1.5 元 /㎡/ 天）的园区
排除剩余租期不足 5 年的园区（避免中途搬迁）
第三步：实地考察 + 数据采集（微观筛选）
对第二步筛选出的 8-12 个园区，进行实地考察，采集以下核心数据，最终确定 8 个最优候选点：
仓库基本信息：面积、层高、柱距、装卸货平台数量、是否有电梯
成本信息：租金、物业费、水电费、停车费、递增率（一般每年 3%-5%）
路网信息：到绕城高速出入口的距离、到城市核心区的平均车程、到各主要商圈的车程
配套信息：周边有没有加油站、维修站、员工宿舍、餐饮
合规信息：产权证明、消防验收报告、环保验收报告

2. 5 年全周期总成本详细拆解
plaintext
5年全周期总成本 =
初始一次性投入（仓库改造、货架、分拣设备、叉车、系统）
+ 5年固定成本（租金、物业费、人工、水电）
- 期末残值（设备二手价值、剩余租期转让价值）
初始改造费：100-200 元 /㎡（三温仓改造）
年运营成本：初始投资的 15%-20%
期末残值：初始投资的 20%-30%
3. 批量计算到门店的距离
不用手动一个个查，用高德 / 百度地图的 Web API，写 10 行 Python 代码就能批量计算所有 RDC 到所有门店的路网距离：
python
运行
import requests
def get_distance(origin, destination, ak="你的高德地图AK"):
url = f"https://restapi.amap.com/v3/distance?origins={origin}&destination={destination}&type=1&key={ak}"
response = requests.get(url).json()
return float(response["results"][0]["distance"]) / 1000  # 转换为公里
四、RDC 候选池规模控制（非常重要）
最终输入模型的 RDC 候选点数量严格控制在 8-12 个
超过 12 个：模型求解时间指数级增长，且大部分候选点本身就没有竞争力
少于 8 个：可能会漏掉真正的最优解
技巧：如果某个物流带有多个园区，只保留租金最低、路网最好的 1-2 个即可。
五、常见坑与避坑指南
不要只看租金，忽略配送成本很多人会选租金最便宜的 RDC，但如果位置偏远，配送成本会大幅增加。比如一个租金便宜 50 万 / 年的 RDC，可能会导致每年配送成本增加 100 万，反而得不偿失。这也是为什么我们要把配送成本纳入模型优化的核心原因。
不要忽略货车限行政策北京很多区域白天禁止蓝牌货车通行，杭州对货车也有严格的限行规定。如果 RDC 在限行区内，只能夜间配送，会大幅增加人工成本和配送时效，影响生鲜品质。
不要租面积过大的仓库社区超市 RDC 的面积需求和门店数量成正比，1 万平米足够服务 50-80 家门店。如果租了 2 万平米的仓库，一半空着，会造成巨大的浪费。
一定要实地考察网上的信息很多是过时的，比如园区已经拆迁、仓库已经租出去了、周边路网已经改道了。实地考察是验证候选点可行性的唯一方法。
